# Prerequisites

Before using cruijff_kit, you should be comfortable with the skills below. Each link points to a free, self-paced tutorial.

## Required Skills

- **Git & GitHub** - Cloning repos, pulling changes, working with branches
  - [Version Control with Git](https://swcarpentry.github.io/git-novice/) (Software Carpentry) - especially Episode 7
- **Unix Shell** - Navigating directories, running commands, editing files
  - [The Unix Shell](https://swcarpentry.github.io/shell-novice/) (Software Carpentry) - Episodes 1, 2, 3, and optionally 6
- **SSH & Remote Work** - Connecting to HPC clusters via SSH
  - [Remote Development via SSH](https://www.digitalocean.com/community/tutorials/how-to-use-visual-studio-code-for-remote-development-via-the-remote-ssh-plugin) (Digital Ocean)
- **Conda & pip** - Creating environments, installing packages
  - [Introduction to Conda for (Data) Scientists](https://carpentries-incubator.github.io/introduction-to-conda-for-data-scientists/) (Carpentries Incubator) - Parts 1 and 2

## Required Accounts

- **HuggingFace** - Account and access token for downloading models
  - [Getting a HuggingFace Token](https://huggingface.co/docs/hub/en/security-tokens)

## Recommended

- **SLURM** - Basic job submission and monitoring
  - [Princeton Research Computing SLURM Primer](https://researchcomputing.princeton.edu/support/knowledge-base/slurm)
- **Claude Code** - Powers the automated experiment workflows (design, scaffold, run)
  - [Claude Code documentation](https://docs.anthropic.com/en/docs/claude-code)
  - Manual workflows are available if you don't have Claude Code access

## Optional

- **Python** - Helpful for reading the codebase or adding functionality
  - [Programming with Python](https://swcarpentry.github.io/python-novice-inflammation/) (Software Carpentry)
